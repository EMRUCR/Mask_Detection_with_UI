Mask_Detection_with_UI 

This is a project to fine-tune the YOLO26n model for face mask detection via webcam. 
The UI was created using PytQ5; 
CUDA and CPU versions without the UI are available in my other repositories. 
Dockerhub links are also included in the readme file.
